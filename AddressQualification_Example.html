Address Qualification Demo

This page demonstrates the address qualification script. The original zip code input will be transformed into an address input with Google Places Autocomplete.

Before the script will work, you need to:

  3. Replace YOUR_GOOGLE_MAPS_API_KEY in AddressQualificationScript.js with your actual key
  4. Load the script on this page (see options below)
